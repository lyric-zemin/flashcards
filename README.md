# Flashcards 学习应用

一个专为汉字学习设计的交互式闪卡应用，融合游戏化元素，让学习变得更加有趣和有效。

## 🎯 项目特色

### 核心功能
- **闪卡学习**：浏览、学习和管理汉字闪卡
- **游戏化系统**：通过游戏模式巩固学习成果
- **成就系统**：完成任务获得成就和积分
- **用户认证**：安全的注册和登录功能
- **个人资料**：查看学习进度和成就

### 游戏模式
1. **连连看 (ConnectGame)**：连接相关的汉字和含义
2. **填空题 (FillGame)**：根据提示填写正确的汉字
3. **配对游戏 (MatchingGame)**：将汉字与对应的图片或含义配对
4. **记忆游戏 (MemoryGame)**：考验记忆力的翻牌游戏
5. **拼图游戏 (PuzzleGame)**：通过拼图方式学习汉字结构
6. **问答游戏 (QuizGame)**：测试汉字知识的问答挑战

### 技术特色
- **前后端分离**：现代化的全栈架构
- **TypeScript**：类型安全的代码
- **React**：响应式前端界面
- **Prisma ORM**：优雅的数据库操作
- **游戏化设计**：积分、成就和排行榜
- **动画效果**：流畅的用户界面动画
- **响应式布局**：适配不同设备

## 🚀 快速开始

### 安装依赖

```bash
# 安装所有依赖
pnpm install
```

### 运行项目

#### 启动后端服务
```bash
pnpm dev:backend
```

#### 启动前端开发服务器
```bash
pnpm dev:frontend
```
